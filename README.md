# 🎮 Hangman Game (Python)

A simple and interactive **text-based Hangman Game** built using Python.  
The player has to guess the hidden word by entering one letter at a time with a limited number of incorrect attempts.

---

## 📌 Features

- 🔤 Random word selection  
- 🎯 Letter-by-letter guessing  
- ❌ Limited attempts (6 wrong guesses)  
- 📊 Displays progress after each guess  
- 🧠 Enhances logical thinking  

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Concepts Used:**  
  - Loops (`while`)  
  - Conditional Statements (`if-else`)  
  - Strings & Lists  
  - Random Module
